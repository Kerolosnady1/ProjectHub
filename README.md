# ProjectHub - Project Manager

ProjectHub is a desktop application built with Python and Tkinter that helps developers organize and manage their projects. It supports features like project categorization, statistics, backup, export/import, dark mode, search, thumbnails, and Git integration.

## Features

- **Add/Edit/Delete Projects**: Store project details (name, path, description, category, GitHub URL, thumbnail, estimated hours).
- **Project Categories**: Classify projects by language or type.
- **Statistics**: View total projects, estimated hours, and category distribution (with chart).
- **Backup & Restore**: Create compressed backups of the database and thumbnails; restore from backup.
- **Export/Import**: Share a single project as a `.projhub` file.
- **Dark Mode**: Toggle between light and dark themes.
- **Search**: Filter projects by name, description, or category.
- **Thumbnails**: Display a thumbnail image for each project.
- **Git Integration**: If a project is a Git repository, show branch, last commit, and uncommitted changes; open GitHub URL directly.
- **Run Projects**: Launch Python scripts or executable files directly from the interface.
